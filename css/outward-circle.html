# 向外填充一个圆

# Method 1

通过background和radial-gradient实现

<div class="circle">

</div>

.circle {
  width: 50px;
  height: 50px;
  border: solid 1px blue;
  margin: 30px auto;
  border-radius: 50%;
  background-image: radial-gradient(blue 50%, transparent 50%); /* 一半蓝色，一半透明，营造填充的过程 */
  background-size: 0 0; 
  background-repeat: no-repeat;
  background-position: 50% 50%; /* radial-radient 从.circle圆心往四周 */
  transition: all 1s; /* 过渡 */
}

.circle:hover {
  background-size: 200% 200%; /* 从 size 0 到 200% */
}
